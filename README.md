# Coding Lessons Portfolio

## Description
This repository contains a collection of my coding classwork and homework assignments, organized into a portfolio website. It showcases exercises and projects in Python programming and web development technologies including HTML, CSS, and JavaScript.

## Features
- **Python Lessons**: Comprehensive exercises covering fundamental concepts such as syntax, functions, control flow, lists, loops, strings, modules, and more advanced topics like classes, inheritance, polymorphism, dictionaries, files, and function arguments.
- **Web Development**: Basic to intermediate examples in HTML, CSS, and JavaScript.
- **Interactive Website**: A navigable website built with HTML, CSS, and JavaScript to display the lessons and projects.
- **Organized Structure**: Lessons are categorized by topic for easy access and reference.
